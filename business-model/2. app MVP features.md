# Funigo app (for client) key features for MVP

- iPhone and Andriod support
- Email to receive one-time passcode to register and login
- Long active authentication token (12 month) because of the fact of mobile device. After expiry, resend OTP via the orignal method (email)
- The UI layout is similar to ChatGPT or other AI chat apps, where
    - On top left, there is a fold/unfold menu to show slide-in menu panel, where
        - projects (chat threads) are listed.
        - at bottom, fixed item is settings icon
            - Clicking settings icon will pop up settings page, where
                - User can set user name (display name)
                - Phone number
                - email (cannot change)
    - The main page is a conversation message list, like WeChat, Whatsapp, and ChatGPT
        - messages can be text, photo, video, documents (PDF).
        - client's messages are aligned on right hand side.
        - other people or roles message are aligned on left hand side.
    - At bottom, fixed message input
